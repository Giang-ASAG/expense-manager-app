# Vi Khon – Smart Expense Manager

Vi Khon is a personal finance management application built with Flutter and Firebase.
It allows users to track daily expenses, manage multiple wallets, and gain insights into their
financial behavior through clear and structured data visualization.

---

## Features

* Track income and expenses in real time
* Manage multiple wallets such as cash, bank accounts, and e-wallets
* Attach receipt images to transactions
* Analyze spending habits with charts
* Secure user authentication with Firebase
* Real-time data synchronization using Cloud Firestore

---

## Tech Stack

* Flutter (Cross-platform UI framework)
* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* MVVM Architecture

---

## Getting Started

### Prerequisites

* Flutter SDK
* Firebase project configuration

### Installation

```bash
git clone
cd your-repo
flutter pub get
flutter run
```

## Firebase Configuration

To run this project, you need to provide Firebase configuration files:

* android/app/google-services.json
* ios/Runner/GoogleService-Info.plist (for iOS)

The application will not run correctly without these files.
